# nAIgios プロジェクト — Claude への指示

## nAIgios 関連の応答フォーマット

nAIgios のアラート・誤検知・スコア・動作について言及する場合は、通常の返答と分離して以下の形式で表示すること。

```
[nAIgios]: <nAIgios に関する説明・補足>

<通常の返答>
```

**悪い例（区別できない）**
> 引き続きスペルチェッカーの固有名詞誤検知です。無視して問題ありません。続きはどうしますか？

**良い例（区別できる）**
> [nAIgios]: 引き続きスペルチェッカーの固有名詞誤検知です。無視して問題ありません。
>
> 続きはどうしますか？
